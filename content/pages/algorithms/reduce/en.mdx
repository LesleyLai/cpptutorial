Consider the following code, where you calculate a 

```cpp
std::vector<double> nums;

...

double sum = 0;
for (double num : nums) {
    sum += num;
}
const double mean = sum / std::size(nums);
```

```cpp
const double mean = std::reduce(nums.begin(), nums.end()) / std::size(nums);
```

```cpp
const double mean = std::reduce(std::begin(nums), std::end(nums)) / std::size(nums);
```
